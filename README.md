## 功能一览

| 功能 | 说明 |
|------|------|
| 多便签 | 可新建任意数量的便签，每张独立 |
| 任务管理 | 添加、勾选、删除任务，支持内联编辑 |

| 颜色主题 | 5 种预设颜色 + 系统色盘自定义，悬停实时预览，每张便签单独设置 |

| 透明度 | 滑块实时调节（30%–100%），持久保存 |
| 多语言 | 支持中文、英语、日语、德语、法语、西班牙语，切换即时生效 |
| 便签列表 | 一键查看所有便签，控制显示 / 隐藏，直接删除 |
| 置顶 | 固定便签始终显示在其他窗口前面 |

| 开机自动启动 | 登录时自动启动（写入注册表，支持 exe 和脚本两种运行方式） |
| 数据持久化 | 关闭窗口只是隐藏，重新打开原样恢复 |
| 单实例唤醒 | 再次双击 exe 可唤醒所有隐藏的便签 |

## 快速开始

直接双击 `SheepNote.exe`，无需安装 Python 环境。

首次启动会在 exe 同级目录下自动创建 `notes_data.json` 保存数据。

## 界面说明

### 工具栏

```
[⠿ SheepNote] [📋] [＋]       [置顶] [🔒] [×]
     标题/拖动  列表  新建       置顶   锁定  隐藏
```

| 按钮 | 功能 |
|------|------|
| `⠿ SheepNote` | 拖拽移动便签位置 |
| `📋` | 打开便签列表与设置面板 |
| `＋` | 新建一张便签 |
| `置顶` | 切换置顶模式 |
| `🔒` | 切换锁定模式（锁定后内容只读） |
| `×` | 隐藏当前便签（数据保留，再次双击 exe 可恢复） |

### 任务区

- **添加任务**：在底部输入框输入内容，按 `Enter` 确认
- **完成任务**：点击左侧 `☐` 勾选，已完成任务显示删除线
- **拖拽排序**：拖动任务左侧 `⠿` 手柄调整顺序（锁定模式下隐藏）
- **编辑任务**：直接点击任务文字进行修改，失去焦点时自动保存
- **删除任务**：鼠标悬停在任务上，点击右侧出现的 `×`，支持 3 秒撤销
- **键盘快捷键**：`Ctrl+D` 切换完成状态、`Ctrl+Delete` 删除任务、`↑↓` 在任务间导航
- **右键菜单**：在任务区右键可清除所有已完成任务 / 删除当前便签

### 调整大小

拖动右下角的 `◢` 三角形调整便签窗口大小（最小 220×150）。

### 滚动条

内容超出便签高度时，右侧会出现细圆角滚动条，静止 2.5 秒后自动隐藏。

### 边缘自动收缩

将便签拖至屏幕任意边缘（8px 以内），停留约 1 秒后自动以 cubic ease-out 动画折叠，保留 16px 可见条。鼠标悬停时展开，移开后 1 秒再次收缩。可在设置面板中开启或关闭此功能。

## 便签列表与设置

点击工具栏 `📋` 打开综合面板：

**便签列表**

- 显示所有便签的名称、任务数量、完成情况
- **显示 / 隐藏**：点击每行右侧按钮切换单张便签的可见状态
- **删除便签**：点击每行右侧 `×` 按钮（仅剩一张时禁用）
- **新建便签**：点击底部 `＋ 新建便签`

**当前便签设置**

- **颜色**：点击色块切换，悬停实时预览，点击 `…` 打开系统色盘
- **字号**：拖动滑块实时调节（9–18pt）
- **透明度**：拖动滑块实时调节（30%–100%）
- **语言**：点击语言缩写切换界面语言，重启后保留

**全局选项**

- **边缘自动收缩**：Apple iOS 样式开关，开启后拖至屏幕边缘自动折叠
- **开机自动启动**（仅 Windows）：Apple iOS 样式开关，开启后注册开机启动项

|------|------|
| 🟡 `#FFF9C4` | 暖黄 |

## 多语言

在便签列表面板底部点击语言缩写即可切换：

| 缩写 | 语言 |
|------|------|
| 中 | 简体中文 |
| EN | English |
| 日 | 日本語 |
| DE | Deutsch |
| FR | Français |
| ES | Español |

切换后界面文字、菜单、提示语即时更新，语言偏好保存在数据文件中。

## 数据文件

数据保存在 exe 同级目录的 `notes_data.json`，格式为 JSON，可手动备份。

```
D:\任意目录\
├── SheepNote.exe     ← 主程序
└── notes_data.json   ← 数据（自动生成）
```

> `notes_data.json` 中保存了每张便签的任务内容、位置、大小、颜色、字号、透明度、语言等信息。删除此文件将清空所有便签数据。

## 从源码构建

**Windows：**
```bash
pip install pyinstaller
python build.py [--bump]
```
构建产物：`dist/SheepNote-v5.1-Windows.exe`

**macOS：**
```bash
pip install pyinstaller pystray pillow
python -m PyInstaller --onefile --windowed --icon=sheep.ico \
    --name=SheepNote-v5.1-macOS sticky_note.py
```
构建产物：`dist/SheepNote-v5.1-macOS.app`

## 技术栈

- **Python 3.12** + **tkinter**（UI）
- **ctypes / Windows API**：单实例 Mutex、IPC Named Event、多显示器工作区检测、DWM 圆角、注册表读写
- **colorsys**：工具栏颜色自动推算
- **winreg**：开机自动启动注册表操作（Windows only）
- **PyInstaller**：打包为单文件 exe
